# Glossary

**In one sentence:** a plain-language decoder for every acronym and term used in the
book - read it once, or flip back whenever a chapter throws a new one at you.

The full version mirrors (and slightly expands) the glossary in the repository
[README](https://github.com/hanishi/pekko-dast#glossary), kept in sync with it.

**This chapter will cover:** one friendly line each for **DAST**, **XSS** (reflected vs
DOM-based), **SQLi**, **SSRF**, **SSTI**, **IDOR**, **LFI / path traversal**, **CORS**,
**JWT**, **CSRF**, **SPA**, **XHR / fetch**, **CDP**, **OAST**, **LLM**, the
**source / sink / taint** trio, **deterministic**, and **probe / gated / observe-only** -
each with a tiny example so the word sticks, not just a definition.
